# zsh-iptv
Z shell functions for IPTV streaming with [mpv](https://mpv.io/)

## Installation
Copy zsh-iptv.zsh into your ~/.zsh configuration directory  
Add the invocation of zsh-iptv.zsh to your .zshrc
```
source ~/.zsh/zsh-iptv/zsh-iptv.zsh
```

## Usage
```
$ tv-3sat
$ tv-arte
```
